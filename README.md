# BananaGames
Hi. BananaGames is a very lightwieght indie game. It can even run on the oppisite of a high-powered pure-tungsten engine with a anti-vibration mount capable of 29 killowatts of electricity of a CPU.
Actually you need two cores or more. You also need 4GB ram or more. 8GB ram is considered the bare minimum if you are using the windows version of app. 16GB is ideal. 32GB+ will be considered OP.
